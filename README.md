# Description
The aim is to create a hobbyist signal generator using the AD9833, with additional custom features such as an LCD screen and control knobs.

 # Features
- adjustable wave shape - sinusoidial, triangle, rectangular
- adjustable frequancy (10Hz - 50kHz)
- adjustable amplitude
- DC offset
- battery powered
# Parts used
- AD9833 signal generator IC
- op amp
- LCD
- microcontroller STM32F103C8T6
# Files Included
- firmware: source code for the microcontroller
# Tools used in the project
- STMCUBEIDE, HAL libraries
